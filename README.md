## 📖 Overview

BIDWARS is an **Agentic AI-powered IPL Auction Simulator** where users participate in a live IPL-style auction against **9 autonomous AI franchise agents**.

Each AI agent independently analyzes players, manages its remaining budget, builds a balanced squad, and dynamically adjusts its bidding strategy based on the current auction state.

The project demonstrates **Multi-Agent AI**, **LLM orchestration**, **Prompt Engineering**, and **Real-Time Decision Making** in an interactive full-stack web application.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/aryangupta3105/BIDWARS.git
```

Move into the project

```bash
cd BIDWARS
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
OPENAI_API_KEY=your_openai_api_key_here
```

Run the project

```bash
npm start
```

or

```bash
npm run dev
```

# 🎮 How It Works

1. Load the IPL player database.
2. Initialize 9 autonomous AI franchise agents.
3. Allocate auction budgets.
4. Present players one by one.
5. AI agents evaluate and place bids strategically.
6. The human player competes against AI in real time.
7. The highest bidder wins the player.
8. Continue until the auction concludes.
